# Docker
```bash
docker pull bkimminich/juice-shop # Owasp Juice Shop lab with web-vulnerabilities
docker run --rm -p 3000:3000 bkimminich/juice-shop # start vuln lab in http://localhost:3000
docker run hello-world # checks if docker works correctly
docker ps # see containers
docker compose up -d # run config file .yml you must be located in file .yml directory
docker compose down # stop all docker labs
docker system prune -f
```
