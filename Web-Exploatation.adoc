== HTTP

Methods which is other than `GET` and `POST`

```bash
HEAD - same as GET but returns headers
PUT - writes something to the specified location
DELETE - deletes something at the specified location
OPTIONS - shows different options in web server
PATCH -  	Apply partial modifications to the resource at the specified location
```
